# AccountBot — Architecture

Public, architecture-only LikeC4 views of the AccountBot systems (the source repos are private).
These diagrams describe component structure, data flow, and design — **no source code, secrets,
or sensitive data**. Two systems:

- **pipegen-agent** — account-scoring & enrichment pipeline.
- **databot-agent** — tool-agnostic client layer for the internal data plane.

Published to GitHub Pages: https://sjarmak.github.io/accountbot-architecture/
